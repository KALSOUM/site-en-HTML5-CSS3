SQUELETTE POUR VOS PAGES WEB

Ne perdez pas votre temps !

Chaque fois qu'on crée une page web il serait hautement inefficace
de retaper chaque fois les balises essentielles pour la structure de la page.
Voici donc un squelette (point de départ pour toutes vos pages web) :


Doctype, les metas déclarations, les balises sémantiques html5 telles que header,
nav, article, aside,footer sont les éléments essentiels d'une page web html5.

Organiser un formulaire : l’élément HTML fieldset l’élément HTML fieldset
va nous permettre de regrouper certains éléments d’un formulaire HTML entre eux,
afin de créer différentes « sections » pour notre formulaire et de le structurer.
Pour donner un nom à chaque section, nous allons utiliser l’élément HTML
legend à l’intérieur de fieldset.
